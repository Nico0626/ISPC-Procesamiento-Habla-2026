# ISPC-Procesamiento-Habla-2026
Portafolio de proyectos de la materia Técnicas de Procesamiento del Habla - TSCDIA
¿Qué espero de esta materia?
Espero aprender cómo funcionan los modelos de procesamiento del habla para reconocer voces o ruidos específicos, y entender cómo se pueden aplicar estas técnicas en la práctica. Me interesa especialmente saber cómo aprovechar estos sistemas desde el rol de analista, pudiendo extraer, procesar y analizar datos de audio para obtener información útil.

Áreas de interés dentro de la Ciencia de Datos:
En lo personal, me interesa mucho el área de deep learning, especialmente todo lo relacionado con redes neuronales.
